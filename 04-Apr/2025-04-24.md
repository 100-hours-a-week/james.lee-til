# 날짜: 2025-04-24

## 새로 배운 내용
### 주제 1: Prometheus
- 모니터링 시스템
- 지표(Metric) 수집에 특화된 도구이다
- CPU 사용량, 메모리, 응답 시간 등
- 이를 시계열 DB 형태로 저장함

### 주제 2: Grafana
- 시계열 데이터 시각화 도구
- Prometheus 등에서 수집한 것을 시각화
- 알람 기능 (예: CPU 90% 넘으면 알림)
- 서버 상태를 모니터링할때 유용함

### 주제 3: Kafka
- 메시지 브로커 시스템
- 다른 서비스 간에 데이터를 실시간 전송하기 위한 허브 역할
- 서비스 간 알아야 하는 정보가 줄어들기에, 확장성이 향상된다
- 또 비동기 처리가 가능해져 규모가 커졌을 때 편리하다
