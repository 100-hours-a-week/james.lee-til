## 진행한 내용
### 주제 1: API 수정
#### 기간 수정사항
- 최근 뉴스는 카드 하나에 1일의 기간으로
- 중간 뉴스는 카드 하나에 1주의 기간으로
- 과거 뉴스는 카드 하나에 1달의 기간으로

#### API Request 수정
- 백엔드에서 AI쪽으로 기간(카드)의 리스트를 전달
- AI쪽에서 기간에 맞게 내용을 채워서 반환

#### 오래된 뉴스 병합
- 카드 하나에 1일이었어도 시간이 지나고 하나당 1달로 변경해야 할 수 있다
- 카드 여럿을 보내면 요약하여 하나로 합치는 새로운 엔드포인트 필요
