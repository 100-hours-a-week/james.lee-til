### 주제 1: 스크래핑 품질 향상
#### 문제
- API Parameter로 날짜를 지정해도, 최신 기사로 편향되는 문제
- 30일치 기사를 요청해도, 1일당 1개씩 기사가 나타나지는 않는다

#### 해결책
- Parameter상으로 날짜를 지정하지 않고, 쿼리 검색어로 지정한다
- "이슈 제목 2025-05-07"처럼 여러 번 검색하는 것이 더 정확
- API 토큰이 훨씬 많이 들지만 현재 더 나은 해결책 없음
